# Rock Paper Scissors 🪨📄✂️

Egy egyszerű, böngészőben futó Kő-Papír-Olló játék tiszta HTML, CSS és JavaScript használatával.

## Funkciók

- Klasszikus Kő-Papír-Olló játék (best of 5 vagy végtelen kör)
- Játékos vs. Számítógép
- Véletlen számítógép választás
- Eredmény kijelzése (Győzelem / Vereség / Döntetlen)
- Pontszám számláló
- Letisztult, modern felhasználói felület
- Reszponzív design (mobilról is jól használható)
- Teljesen kliensoldali – nincs szerver szükséges

## Tech Stack

- **HTML5**
- **CSS3** (animációkkal)
- **Vanilla JavaScript**

## Használat

1. Klónozd a repository-t:

```bash
git clone https://github.com/Snewkovits/rockpaperscissors.git
```

Nyisd meg az index.html fájlt bármely modern böngészőben (Chrome, Firefox, Edge stb.).

A játék azonnal elindul. Nincs telepítés szükséges.
